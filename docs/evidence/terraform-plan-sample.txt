## Context

This file shows a representative `terraform plan` sample for the current root module.
`main.tf` always instantiates both `module.nginx_server_dev` and `module.nginx_server_qa`; the selected `-var-file` only supplies input values.

## Execution

```bash
terraform init
terraform workspace new dev
cp terraform.tfvars.example local.tfvars
# Edit local.tfvars with real values before planning.
terraform plan -var-file=local.tfvars -out=tfplan
terraform show -no-color tfplan
```

The command above still plans both modules because both are declared in `main.tf`.
